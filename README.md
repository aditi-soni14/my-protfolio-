# Portfolio Website (Next.js)

## Project Overview
**Portfolio Website** is a modern, fully responsive personal portfolio built using **Next.js**.  
It showcases my skills, projects, and contact information with a clean UI, smooth navigation, and optimized performance.  
This portfolio is designed to highlight my work as a web developer in a professional and visually appealing way.

**GitHub Repository:** [Portfolio Website](https://github.com/aditi-soni14/my-protfolio-)

---

## Features
- Modern and clean UI  
- Fully responsive design  
- Fast performance with Next.js optimization  
- Smooth page transitions  
- Dedicated sections for About, Skills, Projects & Contact  
- SEO-friendly structure  
- Easy to customize and scale  

---

## Tech Stack
- Next.js (React Framework)  
- React.js  
- JavaScript  
- Tailwind CSS / CSS  
- HTML5  

---

## Getting Started
Follow the steps below to run this project locally:

```bash
# 1. Clone the repository
git clone https://github.com/aditi-soni14/my-protfolio-.git
cd my-protfolio-

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev

# The app will now run on http://localhost:3000/
```
